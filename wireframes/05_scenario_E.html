Data model reference for Zoho CRM implementation. Three modules: Governance Body, Governance Membership (junction), and Competency tags on Contact records.

Field Name API Name Type Required Notes
Body Name Body_Name Text (255) Required Full formal name of governance body
Type Body_Type Picklist Required Board · Committee · Advisory
Purpose Body_Purpose Text Area (2000) Mandate and primary responsibilities
Established Date Established_Date Date Used for tenure duration calculation
Reporting To Reporting_To Lookup Self-referential → Governance Body
Active Member Count Active_Member_Count Integer Auto Rollup from Memberships where Status=Current
Vacancy Status Vacancy_Status Picklist Auto Active · Vacancy — set by workflow on expired term
Competency Gap Count Competency_Gap_Count Integer Auto Formula: Required – Covered
Coverage Rate % Coverage_Rate Percentage Auto Covered / Total × 100
