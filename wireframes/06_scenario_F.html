Module field reference — Scenario F

Field-level model for modules in Scenario F: NC Account, NC Volunteer, Training Record, and MoU Obligation. Every API name maps to the Zoho CRM implementation.

NC Account module

Field labelAPI nameTypeNotes
NC NameNC_NameTextFull name, e.g. "UWC Kenya"
CountryCountryPicklistISO country list; drives regional grouping
StatusNC_StatusPicklistActive · Suspended · Observer · Inactive
NC ChairNC_Chair_LookupLookup → NC VolunteerOne active Chair per NC
MembersMember_CountInteger · AutoRoll-up from NC Volunteer records
EstablishedEstablished_YearIntegerYear NC was formally established
Governance BodyGovernance_Body_LookupLookup → Governance BodiesRegional advisory body link
MoU HealthMoU_HealthFormulaGood / Attention / Critical — computed from overdue MoU Obligations roll-up

Field labelAPI nameTypeNotes

Portal AccessPortal_RolePicklistChair · Member · None — drives NC Portal permissions
