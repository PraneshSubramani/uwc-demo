Data-level scoping is enforced by Roles, not Profiles

The profile permissions above control what modules a user can touch. Which records they see within those modules (Ana → Brazil only, Marcus → Germany only) is enforced by the Role hierarchy + Data Sharing Rules in CRM Setup → Security → Data Sharing Settings. Confirm the data sharing rules there to enforce NC-level row-level isolation.
