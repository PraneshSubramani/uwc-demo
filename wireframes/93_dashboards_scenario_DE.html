How to build it

1 Custom HTML widget Widget

WhereSetup → Customisation → Widgets → + Create → "Governance Matrix"
Hosted onA Zoho CRM Dashboard named "Governance Competency Matrix"
RowsActive Boards (Body_Type = Board AND Body_Status = Active)
ColumnsCompetencies where Required_Board = true
Cell logicgreen tick if ≥1 Active membership in that body holds the competency, else red cross
Footer rowGap_Count per competency (from the Competencies module)

2 Data feed — inline Deluge Deluge

The widget fetches JSON from an inline function getCompetencyMatrix(). Gap_Count is precomputed nightly (Zoho's native roll-ups can't express "boards missing this competency"):

recomputeCompetencyGaps() — daily @ 02:00 UTC

allComps  = zoho.crm.getRecords("Competencies", 1, 200);
allBodies = zoho.crm.searchRecords("Governance_Bodies",
  "(Body_Type:equals:Board)and(Body_Status:equals:Active)");
for each c in allComps {
  gap = 0;
  for each b in allBodies {
    members = zoho.crm.searchRecords("Governance_Memberships",
      "(Governance_Body:equals:"+b.get("id")+")and(Membership_Status:equals:Active)"
      +"and(Competencies_Held:in:"+c.get("id")+")");
    if (members.size() == 0) gap = gap + 1;
  }
  zoho.crm.updateRecord("Competencies", c.get("id"), {"Gap_Count": gap});
}

Pre-seed before the live demo

Seed Gap_Count manually the evening before — the scheduler only fires overnight, so a fresh build would otherwise show zero gaps on stage. Target: widget loads in under 2 seconds and renders ≥ 4 red cells.
