Module field reference — Scenario C

Field-level model for modules touched in Scenario C: Applications (review workflow fields), Review Scores subform, and Safeguarding Cases module. Every API name matches the live Zoho org.

Applications module — Review workflow fields

Field labelAPI nameTypeNotes
Review CountReview_CountInteger · AutoIncremented by workflow on each Review Score record creation
All Reviews CompleteAll_Reviews_CompleteBoolean · AutoSet TRUE when Review_Count = total assigned reviewers; unlocks score visibility for NC Admin
Safeguarding Flag StatusSafeguarding_Flag_StatusPicklistNone · Under Review · Escalated · Resolved · Closed. When != None: stage locked, emails fired
StageStagePicklistSubmitted → Under Review → Shortlisted → Panel → Nominated → Placed
Assigned ReviewersAssigned_ReviewersMulti-user LookupLinked to Users; drives reviewer queue filter and blind-review enforcement
